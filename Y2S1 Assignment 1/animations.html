Animations used on this site

This page explains and demonstrates the CSS animations used across the website. Each demo below is a minimal example of the exact animation used elsewhere, with a short note on where you can see it live.

1) Navigation link underline

The navigation links have an animated underline that expands from left to right on hover/focus or when the link is active. This is implemented in the header using a pseudo-element with a width transition.

2) Heading slide-in

Hero headings slide in from the left and fade in on load using the slideIn keyframes defined in the stylesheet. Example below uses the same structure as the Home page hero.

3) Image/content fade-in

Images and certain blocks fade in and move up slightly using the fadeIn keyframes defined in the stylesheet. The demo below uses the same .profile class that fades in on load.

4) Scrolling skills ticker

The skills list on the Home page scrolls horizontally in a loop using the loop keyframes. This demo reuses the exact same structure and classes.
