Enhancements

Animations

We added animations into some of the objects within the webpages. For example, the company logo on the top left which eases in and out when you hover over it with the mouse. There are also some intro animations for the main titles.


  • Effect: When a user hovers over the logo, it slightly enlarges (zooms in) to 110% of its original size.
  • Purpose: This effect draws attention to the logo, encouraging user interaction and signaling that the element is clickable.

Heading animation code

  • Effect: Moves an element from off-screen on the left while simultaneously changing its opacity from 0 to 1.
  • Purpose: Creates an engaging entrance effect that captures user attention when elements appear.
